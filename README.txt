In this folder are the exercises for the Python API homework for the Rice Bootcamp, June 2020.

There are sub folders containing relevant code:
1. WeatherPy:  contains WeatherPy_Ross_Davidson.ipynb, which is the primary code to run for this exercise.
2. VacationPy: contains VacationPy_Ross_Davidson.ipynb, which is the primary code to run for this exercise.
3. output_data: contains output printed by the primary code.
4. Working_Files: practise code and notes

The primary code reaches for api_keys.py. This is will need to be updated with your api keys for the code to run.
